Projetos

Projeto 1

Objetivo do Projeto:

O objetivo deste projeto é criar uma aplicação web simples que permite aos usuários gerar cores aleatórias em formato hexadecimal. Esta aplicação serve como uma ferramenta útil para designers, desenvolvedores e entusiastas de cores, proporcionando uma forma rápida e fácil de obter novas cores para seus projetos.

Projeto 2

Objetivo do Projeto:

O objetivo deste projeto é criar uma aplicação web simples que permite aos usuários gerar senhas aleatórias seguras. A aplicação ajuda a criar senhas fortes que são difíceis de adivinhar, aumentando a segurança das contas dos usuários.
